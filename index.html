About This Page

What is this?

An interactive dashboard of known studio closures and workforce reductions across the gaming industry from 2022-2025.

Source Code on Github

Built as a demonstration of web-based data visualization and AI-assisted data product development — the initial version was created and deployed in less than a day with the help of Claude Code.

The data

All records are sourced from the Game Industry Layoffs archive, meticulously collated by @dekaf. Each entry includes studio name, announcement date, headcount where known, parent company, studio type, and location.
